# 离线阅读器

这是一个基于 TypeScript、React/Vite 和 Capacitor 构建的离线阅读器应用程序，旨在为用户提供便捷的跨平台阅读体验。

## 功能
- **离线访问**: 用户可以下载文章，在没有网络连接的情况下进行阅读。
- **跨平台支持**: 支持在 iOS 和 Android设备上使用，还可以在网页上进行访问。
- **快速反应**: 采用 Vite 构建，确保快速的加载时间和流畅的用户体验。
- **自定义主题**: 用户可以选择不同的主题，个性化他们的阅读体验。
- **书签功能**: 允许用户对喜欢的文章进行书签，方便后续查看。

## 技术栈
- **TypeScript**: 提供类型安全，提升开发效率。
- **React**: 用于构建用户界面的 JavaScript 库。
- **Vite**: 下一代的前端构建工具，提供快速的开发服务器。
- **Capacitor**: 用于将 web 应用程序变成原生手机应用的工具。

## 安装
1. 克隆该项目到本地：
   ```bash
   git clone https://github.com/fyu0h/OfflineReader.git
   ```
2. 安装依赖：
   ```bash
   cd OfflineReader
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
